#### **Password**

填入正确的密码即可直接使用，可[联系](mailto:wyf9@wyf9.top)获取.

#### **Get api key**

可以在设置中使用自己的 API 和 API key, 也可以使用 ChatNio 的免费模型:

1. 打开 [ChatNio](https://chatnio.net) 官网
2. 注册或登录一个账号
3. 点击左下角的三点或右上角的头像, 打开菜单
